# GD Physics
This mod brings physical movements to <cg>Menu Layer</c> and <cy>Creator Layer</c>. All nodes now feature **gravity**, **collisions**, and **dynamic bouncing**.

# How to use
* **Settings:** Settings can be found at button in the upper right corner of the screen.
* **PC Users:** Right-click and hold on any node to drag it around.
* **Mobile Users:** Tap the **Setting** button in the top right corner and select <cl>Enable Drag</cl>.

# Mod Settings
- You can customize whether to enable physics for the <cg>Menu Layer</c>, <cy>Creator Layer</c>, or both and can customize the Gravity, Bounceness.
